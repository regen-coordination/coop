## Medium Findings

### M1. ZK proof generation hardcoded to `null` [DEFERRED, 4 cycles]
- **File**: `packages/extension/src/background/handlers/review.ts:66`
- **Issue**: Anonymous publish sets `membershipProof = null` with TODO. Snarkjs WASM workers require `URL.createObjectURL` and `new Worker`, both unavailable in MV3 service workers.
- **Status**: DEFERRED. The fix requires either (a) offscreen document delegation for ZK operations, or (b) a different ZK library compatible with MV3 constraints.
- **Rationale**: This is a structural platform limitation, not a code quality issue. Anonymous publishing still works -- it just skips the proof. Stops escalation.

### M2. `PopupDraftListItem` type drift from `ReviewDraft` [STILL OPEN, 3 cycles]
- **File**: `packages/extension/src/views/Popup/PopupDraftListScreen.tsx:74,82`
- **Issue**: `draft as unknown as ReviewDraft` casts used to bridge between `PopupDraftListItem` (UI type) and `ReviewDraft` (domain type). The types don't structurally overlap.
- **Impact**: If `ReviewDraft` shape changes, these casts silently break at runtime.
- **Recommendation**: Either make `PopupDraftListItem extends Pick<ReviewDraft, ...>` or create a shared adapter function.

### M3. `useI18n.tsx` production `as any` usage [NEW]
- **File**: `packages/app/src/hooks/useI18n.tsx:40`
- **Issue**: `let value: any = (translations as any)[language]` uses double `as any` for nested translation lookup. Has biome-ignore comment.
- **Impact**: Type safety gap in the i18n path. If translation keys are mistyped, no compile-time error.
- **Recommendation**: Type `translations` with `Record<LanguageCode, Record<string, unknown>>` or use a typed i18n library.

### M4. `agent-models.ts` production `as any` usage [NEW]
- **File**: `packages/extension/src/runtime/agent-models.ts:215`
- **Issue**: `(pipeline as any)('text-generation', ...)` casts transformers pipeline to bypass overload resolution. Has biome-ignore comment.
- **Impact**: Low practical risk (transformers library types are complex), but masks potential API changes.
- **Recommendation**: ACCEPTED. The transformers.js pipeline types are genuinely unrepresentable -- the biome-ignore is appropriate.

### M5. `useScrollHijack.ts` is dead code [NEW]
- **File**: `packages/app/src/hooks/useScrollHijack.ts`
- **Issue**: Exported but never imported anywhere in the codebase. knip confirms unused. The file also triggers the only remaining lint error (missing `elementRef.current` in useEffect dependency array).
- **Recommendation**: Delete the file. It was a demo utility that is no longer used.

### M6. `permissionless` flagged as unused dep in extension [FALSE POSITIVE]
- **File**: `packages/extension/package.json`
- **Issue**: knip flags `permissionless` as unused in extension, but it's consumed through `@coop/shared` which imports `permissionless/accounts` and `permissionless/clients`. The extension may need it as a peer/hoisted dep for the wxt build plugin at `wxt.config.ts:69`.
