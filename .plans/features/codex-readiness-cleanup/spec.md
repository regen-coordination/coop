## Summary

Keep a small backlog of repo-hygiene work that materially improves agent reliability, but limit it to current hotspots rather than the old repo-wide decomposition campaign.

## In Scope

- narrow cleanup of current large or high-churn files that still hurt agent context
- tracked junk removal and low-risk hygiene with immediate workflow payoff
- small type or lint corrections in the same touched slices

## Out Of Scope

- stale audit counts treated as a literal todo list
- already-finished decompositions
- broad refactors with no clear safety or workflow payoff
